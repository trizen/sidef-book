# [Real constants and functions][1]

```ruby
Num.e     # e
Num.pi    # pi
x.sqrt    # square root
x.log     # natural logarithm
x.log10   # base 10 logarithm
x.exp     # e raised to the power of x
x.abs     # absolute value
x.floor   # floor
x.ceil    # ceiling
x**y      # exponentiation
```
