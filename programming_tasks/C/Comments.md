# [Comments][1]

Single line comment

```ruby
# this is commented
```


These may also be at the end of a line

```ruby
var i = 1; # this is the comment part
```


Embedded comments

```ruby
var distance #`{in meters} = (30 #`{meters} * 100 #`{seconds});
say distance; # prints: 3000
```


Multi-line comments

```ruby
/*
    This is a multi-line comment
*/
```
