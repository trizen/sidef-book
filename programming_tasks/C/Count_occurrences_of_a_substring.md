# [Count occurrences of a substring][1]

Built-in:

```ruby
say "the three truths".count("th")
say "ababababab".count("abab")
```


User-created function:

```ruby
func countSubstring(s, ss) {
    var re = Regex(ss.escape, 'g')      # 'g' for global
    var counter = 0
    while (s =~ re) { ++counter }
    return counter
}
```
