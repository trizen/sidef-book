# [Case-sensitivity of identifiers][1]

```ruby
var dog = 'Benjamin';
var Dog = 'Samba';
var DOG = 'Bernie';
say "The three dogs are named #{dog}, #{Dog}, and #{DOG}.";
```

#### Output:
```
The three dogs are named Benjamin, Samba, and Bernie.
```
