# [Find adjacent primes which differ by a square integer][1]

```ruby
var p = 2
var upto = 1e6
 
each_prime(p.next_prime, upto, {|q|
    if (q-p > 36 && is_square(q-p)) {
        say "#{'%6s' % q} - #{'%6s' % p} = #{'%2s' % isqrt(q-p)}^2"
    }
    p = q
})
```
