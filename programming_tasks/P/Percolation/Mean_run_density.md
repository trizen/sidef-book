# [Percolation/Mean run density][1]

```ruby
func R(n,p) {
    n.of { 1.rand < p ? 1 : 0}.sum
}
 
const t = 100
say ('t=', t)
 
Range(.1, .9, .2).each { |p|
    printf("p= %f, K(p)= %f\n", p, p*(1-p))
    [10, 100, 1000].each { |n|
        printf (" R(n, p)= %f\n", t.of { R(n, p) }.sum/n / t)
    }
}
```
