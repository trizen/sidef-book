# [Generic swap][1]

```ruby
func swap(Ref a, Ref b) {
    var tmp = *a
    *a = *b
    *b = tmp
}
```


or:

```ruby
func swap(Ref a, Ref b) {
    (*a, *b) = (*b, *a)
}
```


or:

```ruby
func swap(Ref a, Ref b) {
    [*a, *b] » (b, a)
}
```


The swap functions must be called with variable references.

```ruby
var (x, y) = (1, 2)
swap(\x, \y)
```
